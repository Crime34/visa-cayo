
# **Système de Visa – GTA RolePlay**

## **Présentation du projet**

Ce projet est un **système de gestion de visa pour serveur GTA RolePlay**, développé dans un cadre **personnel et éducatif**.
Il permet de gérer les demandes d’accès des joueurs via une **interface web dédiée**, facilitant ainsi le travail de l’équipe administrative et améliorant l’expérience des nouveaux arrivants.

L’objectif principal est de proposer un **outil fiable, simple d’utilisation et adapté aux besoins des serveurs RP modernes**.

---

## **Objectifs**

* **Semi-automatiser le traitement des candidatures**
* **Réduire la charge de travail administrative**
* **Améliorer l’organisation et la traçabilité**
* **Offrir une expérience utilisateur claire et fluide**

---

## **Technologies utilisées**

* **PHP**
* **HTML / CSS**
* **JavaScript**

---

## **Fonctionnalités**

* **Enregistrement des visas**
* **Interface d’administration**
* **Sécurisation minimale des données**
* **Système adaptable à tous types de serveurs RP**

---

## **Installation**

1. **Cloner le dépôt :**

```
git clone https://github.com/tonpseudo/visa-cayo.git
```

2. **Déplacer le projet dans le dossier du serveur web** (htdocs, www, etc.)

3. **Configurer la base de données si nécessaire**

4. **Démarrer le serveur local**

5. **Accéder au projet via :**

```
http://localhost/
```
